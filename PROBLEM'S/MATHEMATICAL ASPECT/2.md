# PROBLEM (2): Even Fibonacci Numbers 

## Mathematical Pattern Optimization ($O(\log N)$ Time)
Instead of generating every single Fibonacci number anc checking if it is even, we can use structural properties of the sequence to skip odd terms entirely.

### The Parity Pattern 
Let's analyze the properties of the standard Fibonacci sequence:
* $F_1 = 1$ (Odd)
* $F_2 = 2$ (**Even**)
* $F_3 = 3$ (Odd)
* $F_4 = 5$ (Odd)
* $F_5 = 8$ (**Even**)
* $F_6 = 13$ (Odd)
* $F_7 = 21$ (Odd)
* $F_8 = 34$ (**Even**)

Because $\text{Odd} + \text{Odd} = \text{Even}$ and $\text{Even} + \text{Odd} = \text{Odd}$, **every third Fibonacci number is guaranteed to be even**.
