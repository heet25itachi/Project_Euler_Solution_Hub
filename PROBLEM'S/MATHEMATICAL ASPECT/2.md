# PROBLEM (2): Even Fibonacci Numbers 

## Mathematical Pattern Optimization ($O(\log N)$ Time)
Instead of generating every single Fibonacci number anc checking if it is even, we can use structural properties of the sequence to skip odd terms entirely.

### The Parity Pattern 
Let's analyze the properties of the standard Fibonacci sequence:
* $F_1 = 1$ (Odd)
* $F_2 = 2$ (**Even**)
* $F_3 = 3$ (Odd)
* $F_4 = 5$ (Odd)
* $F_5 = 8$ (**Even**)
* $F_6 = 13$ (Odd)
* $F_7 = 21$ (Odd)
* $F_8 = 34$ (**Even**)

Because $\text{Odd} + \text{Odd} = \text{Even}$ and $\text{Even} + \text{Odd} = \text{Odd}$, **every third Fibonacci number is guaranteed to be even**.

### Deriving the Recurrence Relation
To skip the odd terms entirely, we have to find an algebric link between consecutive even terms. Let's look at three consecutive even terms from the sequence: $2$, $8$, and $34$.
If we look at the standard Fibonacci recurrence rule ($F_n = F_{n-1} + F_{n-2}$), we can expand it backwards:

$$F_n = F_{n-1} + F_{n-2}$$
$$F_n = (F_{n-2} + F_{n-3}) + F_{n-2} = 2F_{n-2} + F_{n-3}$$
$$F_n = 2(F_{n-3} + F_{n-4}) + F_{n-3} = 3F_{n-3} + 2F_{n-4}$$
$$F_n = 3(F_{n-4} + F_{n-5}) + 2F_{n-4} = 5F_{n-4} + 3F_{n-5}$$

By substituting further to align purely with every third index (the even numbers), we arrive at this direct relationship for consecutive even terms:
$$E_n = 4 \times E_{n-1} + E_{n-2}$$

Let's test this with our real values:
$$(4 \times 8) + 2 = 34$$

This exact pattern holds true for every single even number in the sequence. By using this derived formula, your program performs **three times fewer operations** and bypasses all condition evaluation checks, maximizing execution speed.
